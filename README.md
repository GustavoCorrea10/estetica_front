# Thalita Estética 💆‍♀️

Site profissional desenvolvido para a esteticista **Thalita Pinheiro**, com foco em beleza facial e bem-estar. O objetivo foi criar uma presença online moderna, elegante e totalmente adaptada para celular.

---

## 🛠️ Tecnologias

- **HTML5** — estrutura semântica e acessível
- **CSS3** — design responsivo com variáveis, animações e layout grid/flexbox
- **JavaScript** — menu mobile, scroll reveal, contador animado e navegação ativa
- **Font Awesome** — ícones
- **Google Fonts** — tipografia (Cormorant Garamond + DM Sans)

---

## ✨ Funcionalidades

- **Design responsivo** — funciona perfeitamente em celular, tablet e desktop
- **Paleta de cores personalizada** — rosa vinho sofisticado com tons creme
- **Menu mobile** com hamburger animado
- **Seção Serviços** com cards detalhados de cada procedimento
- **FAQ por categorias** — Limpeza de Pele, Microagulhamento, Drenagem Facial, Design de Sobrancelhas e Geral
- **Seção de Cuidados Pós-Tratamento** em accordion
- **Agendamento direto pelo WhatsApp** — sem formulário, sem complicação
- **Seção Instagram** com link para o perfil
- **Botão WhatsApp flutuante** com animação pulse
- **Mapa do Google Maps** incorporado
- **Scroll reveal** com animação suave ao rolar a página
- **Favicon** personalizado
